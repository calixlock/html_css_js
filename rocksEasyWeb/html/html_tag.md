### 태그의 분류

- 블록(block)레벨 요소

  - 한 줄을 다 차지
  - 가로의 한블록을 모두 차지
  - 종류
    - h1, h2 ... h6 (heading제목)
    - p (paragraph 문단)
    - ul, ol, dl (리스트)
    - li (리스트 항목)
    - table (표)
    - div (박스, 의미없음)
    - header (내용의 헤드)
    - section (내용의 덩어리)
    - article (단일 내용, 기사 등)
    - aside (좌우배치되는 요소들)
    - footer(하단 반복 요소들)
    - nav (navigation 사이트 메뉴 그룹)

- 인라인(inline)레벨 요소
  - 종류
    - a (anchor, 링크)
    - span (소단위 글씨, 의미없음)
    - i (태그)
    - input
    - label
    - b
    - em(emphasize 기울임 강조 / 검색 시 수집됨 )
    - strong(강조 더강조 / 검색 시 수집됨)
    - abbr (약어 abbreviation)
    - dfn (전문용어 부연 설명)
