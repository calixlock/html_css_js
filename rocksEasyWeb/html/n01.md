### html tag 구조

```html
<!-- element = 요소 -->
<a href="http://www.naver.com">content</a>
<태그 속성=값>콘텐츠</태그>
```

- tag = 태그
- attribute = 속성
- value = 값
- content = 콘텐츠 = 내용

- html을 보기 위해서 > browser 필요 >

  - 타입형식을 브라우저가 이해하는 방식으로 정확히 전달
    - DTD : DocType Declaration

- deplicated = 사장된 태그
  - font...
- indentation = 들여쓰기
